# Meador
Simple scripting programming language implemented in Java.

Already implemented features:
1. `switch` operator
2. Custom `data structures`
3. Conditional expressions
    1. Boolean expressions
    2. Conditional operators
4. Unary operators
    1. Postfix
    2. Prefix
5. `for` loop operator
6. `string` type
    1. `string` literal
    2. concatenation
